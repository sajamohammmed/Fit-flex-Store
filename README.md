# Fit-flex-Store

أعضاء الفريق: 
سجى محمد رسمي أبو العنين. 
هديل محمد حامد الهنداوي .

أولاً : الهدف من المشروع : تطوير نظام الكتروني لإدارة متجر مستلزمات الجيم ، بهدف تسهيل عملية عرض المنتجات الرياضية ، وإتاحة الفرصة للمستخدمين لتصفح وشراء المعدات الرياضية بسهولة وكفاءة . 

ثانياً : فكرة المشروع ووصف مختصر : نظام متكامل لمتجر رياضي إلكتروني (fitflex) يتيح للمستخدمين استعراض المنتجات ، إضافتها للسلة التسوق واتمام عملية الشراء . 
يعتمد النظام على واجهة مستخدم بديهية ونظام إدارة خلفي (Back-end) باستخدام PHP وMySQL لإدارة العمليات الأساسية (CRUD) للمنتجات والطلبات. 
  

ثالثاً : المستخدمون (Users):
المسؤول (Admin): لديه صلاحيات كاملة لإدارة النظام. 
المستخدم العادي (User): لديه صلاحيات تصفح المتجر وإدارة سلة تسوقه الشخصية. 

رابعاً : صلاحيات كل مستخدم : 
صلاحيات المسؤول (Admin): 
  => إدارة المنتجات (إضافة، تعديل، حذف المنتجات في المتجر).
  => الاطلاع على جميع طلبات الشراء الواردة من المستخدمين.

صلاحيات المستخدم العادي (Customer) 
     =>  إنشاء حساب جديد وتسجيل الدخول.
     =>  تصفح المنتجات الرياضية المتاحة.
     =>  إضافة المنتجات إلى سلة التسوق وإزالتها.
     =>  إتمام عملية الشراء (Checkout) وتأكيد الطلبات.
     =>  تعديل بيانات الملف الشخصي.
